推論規則
$\ \ \ \ t_1 \in T\ \ \ \ t_2 \in T \ \ \ \ t_3 \in T$ 
-----------------------------
$\ \ \ \mathtt{if\ t_1\ then\ t_2\ else\ t_3 \in \mathcal{T}}$

$\{ t_1 \in T,\ t_2 \in T, \ t_3 \in T\}\rightarrow \mathtt{if\ t_1\ then\ t_2\ else\ t_3 \in \mathcal{T}}$
と書く。
